# V1/V2 精细化打磨 · 正式定稿
> 定位目标：把当前 Demo 版打磨成可正式上线基础剪辑器
> 归档日期：2026-05-14

## 一、优先级分级规则
- **P0**：上线阻塞项，不做完不能发正式版，必须优先闭环
- **P1**：体验质感项，P0 全部做完再做，可作为上线后小幅迭代补齐
- 严格按顺序串行开发，**不打乱依赖、不跳任务**，避免返工和状态错乱

---

## 二、正式开发排期（固定，按顺序逐条执行）

### 第一阶段：P0 上线必做

#### T10 · Toolbar 半成品入口隐藏治理
**需求描述**
现有底部工具栏存在未完工半成品入口，容易误导用户。

**规则**
- 已完整闭环功能才显示：修剪、分割、素材替换、字幕、调色/滤镜、转场
- 所有半成品、未开发完成功能全部隐藏入口
- 后续开发完成再放开，绝不放半成品给用户用

**验收标准**
1. 工具栏干净无冗余未完成入口
2. 现有可用功能入口正常可点击、可正常跳转面板

---

#### T1 · 修复片段手柄最大范围溢出黑屏遗留 Bug
**需求描述**
视频/Image3D/ImageMotion 带原生时长素材，右侧手柄仍可无限向外拖拽超出素材时长，超出区域播放黑屏。

**规则**
- 带 `nativeDuration` 素材：右侧拖拽过程中实时硬卡位，禁止拖出边界
- 只能向内缩，不能向外越界
- 静态图/字幕/贴纸无时长限制，可自由拉伸

**验收标准**
1. 视频/3D图片/动态图片 右边界拖到素材边缘直接卡死，拉不动
2. 不会超出时长、播放无黑屏
3. 原有手势不漂移、不跳变保留，无回归问题
4. 松手不回弹，拖拽中途就直接拦截

---

#### T3 · 转场/滤镜/调色 统一渲染稳定性加固
**需求描述**
UnifiedCompositor 多效果共存偶现闪烁、状态残留、删除后渲染链路不回落。

**规则**
- 转场/调色/滤镜可同时叠加正常渲染
- 删除转场/滤镜后自动切回纯预览通路，无黑屏、无残留效果
- 无任何效果时自动走性能短路，不额外消耗性能

**验收标准**
1. 反复增删转场、开关滤镜无闪烁、无黑屏
2. 预览与导出画面效果完全一致无色差
3. 长项目多片段叠加不内存泄漏、不崩溃

---

#### T12 · 全场景 Undo/Redo 边界 Case 补齐
**需求描述**
增删片段、转场、调色、字幕、替换素材 所有操作都支持撤销重做，边界不崩状态。

**验收标准**
1. 全功能操作都可正常 undo/redo
2. 连续多次撤销重做不崩溃、不 UI 错乱
3. 到顶部/底部操作自动禁用按钮，不越界报错

---

#### T7 · 素材替换全链路能力落地（含 3D/动效保留）
**需求描述**
支持选中任意视频/图片片段，从相册替换新素材。

**规则**
1. 替换后**保留原有时长、修剪区间、调色/滤镜效果**
2. 支持两种替换方式：
   - 菜单点击「替换素材」唤起相册
   - 相册素材手动拖拽到片段上直接覆盖替换
3. 特殊类型兼容：
   - Image3D / ImageMotion 替换普通图片后**保留原有 3D/动态动画效果**
4. 替换后时间轴结构、转场、字幕不打乱、不丢失

**验收标准**
1. 视频、普通图片、3D/动态图均可正常替换
2. 替换后所有已加效果、修剪范围全部保留
3. 3D/动效类型换图后动画不消失
4. 不破坏前后转场和轨道排布

---

#### T11 · 草稿自动保存 + 项目恢复健壮化
**需求描述**
新建项目、自动保存、异常退出项目恢复、草稿缩略图生成。

**验收标准**
1. 编辑过程自动定时保存
2. 闪退/杀后台后重新进入可恢复编辑中项目
3. 草稿列表显示项目缩略图、时间、名称

---

### 第二阶段：P1 体验精修（P0 全部完工后启动）

#### T4 · 视频轨道片段缩略图预览
**需求描述**
当前轨道只有颜色块，无视频缩略图，用户无法直观识别片段内容。

**规则**
- 主轨道视频/3D/动态图 内部生成等间距缩略帧
- 时间轴缩放时缩略图自适应密度，不卡顿、不重叠
- 缩略图异步生成，不阻塞 UI

**验收标准**
1. 每个视频片段内部清晰展示缩略预览
2. 缩放、滚动缩略图加载流畅无闪烁
3. 替换素材后缩略图自动刷新

---

#### T2 · 全轨道手势手感统一精修
**需求描述**
时间轴缩放、滚动、片段拖拽、分割、吸附、多次往复修剪，手感不一致、偶尔错位、吸附不精准。

**规则**
- 缩放平滑、惯性滚动自然
- 片段交换、吸附对齐精准无偏移
- 多次连续拖拽/修剪无坐标漂移、无状态错乱

**验收标准**
1. 任意连续操作无卡顿、无跳位置
2. 多片段密集排列拖拽交换不错乱
3. 缩放+滚动联动流畅，无卡死、无手势冲突

---

#### T6 · 空状态/加载中/异常状态 UI 兜底
**需求描述**
无片段、空轨道、素材加载失败、解析失败无友好兜底界面。

**验收标准**
1. 空项目、空轨道有占位提示文字
2. 素材加载中显示 loading，失败显示重试/占位图
3. 异常状态不崩溃、不卡死时间轴

---

#### T8 · 单条字幕完整样式编辑
**需求描述**
完善单字幕：改文字、字体、大小、颜色、描边、阴影、对齐。

**验收标准**
1. 所有样式参数可实时预览
2. 支持 undo/redo
3. 拉伸时长、复制删除正常无错乱

---

#### T9 · 字幕轨道批量统一样式能力（依赖 T8）
**需求描述**
支持选中整条字幕轨道，**批量统一修改轨道内所有字幕**：字体/颜色/大小/描边/阴影。

**规则**
- 批量设置一键应用到轨道全部文字片段
- 批量修改后保留个别单独手动微调的差异化设置
- 支持批量重置恢复默认

**验收标准**
1. 改轨道样式，全部字幕同步生效
2. 单个手动改过的字幕不受批量覆盖冲突
3. 操作可撤销可重做

---

#### T5 · 音频轨道波形可视化
**需求描述**
音频轨道只有空白色块，无波形，无法直观看节奏、卡点。

**规则**
- 音频资源自动解析生成波形图
- 波形颜色、高度适配轨道行高
- 缩放同步适配密度

**验收标准**
1. 导入音频立刻显示完整波形
2. 裁剪分割后波形同步跟着变化
3. 缩放流畅不模糊、不错乱

---

## 三、核心依赖锁死

1. 必须先补全 **Undo/Redo**，再做素材替换、字幕编辑，所有操作必须可撤销
2. 先修 **手柄溢出 + 渲染稳定性**，再做上层体验功能，底层不稳上层都是白做
3. 单条字幕样式（T8）必须在轨道批量样式（T9）之前完成
4. 先收敛 Toolbar 半成品入口，全程不暴露未完工功能，保持版本整洁

## 四、执行规则

1. 严格按这个顺序，**不跳任务、不私自插新功能**
2. 每条任务开发完，按上方「验收标准」自测，再提交复测
3. 每条不过验收标准，不进入下一条，不凑合、不遗留隐性 Bug
4. 所有 V3 高阶功能（多轨道、曲线调色、LUT、海量特效等）**全部暂缓、入口隐藏**，等 V1/V2 正式上线稳定后再规划
