# V1 补齐阶段 P0 开发任务清单

## 总体原则

暂时搁置拉伸交互细节、UI 重叠、自动滚动等体验类优化；
优先补齐轨道基础元素 + 基础编辑能力，先把 V1 主干流程闭环可用。

---

## 一、P0 任务组：轨道基础元素补齐

### 任务1：字幕/文字片段 — 轨道纯色块替换为实时文字预览

**需求描述**
1. 取消字幕、文字片段在轨道上的纯占位色块
2. 轨道片段内实时渲染显示字幕原文内容
3. 简易还原字体、字号缩略视觉效果
4. 定义并固定视频轨、字幕轨、文字轨、音频轨层级顺序与上下排布规则

**验收标准**
- 轨道上能直接看清字幕文字，不再是空白色块
- 字体缩略效果可辨识，不错乱、不溢出
- 多轨道层级固定不乱序，间距统一

### 任务2：音频片段 — 实现波形可视化，替换纯色块占位

**需求描述**
1. 移除音频轨道纯色块占位样式
2. 接入音频振幅解析，轨道内渲染标准音频波形图
3. 固定音频轨道行高，与视频/字幕轨道间距、圆角风格统一

**验收标准**
- 导入音频后轨道正常显示波形线条
- 波形比例正常、不拉伸、不错乱
- 音频轨道视觉样式和其他轨道统一

### 任务3：全轨道统一视觉规范

**需求描述**
统一所有轨道全局样式：
- 轨道固定行高、轨道之间上下间距
- 片段圆角、内边距
- 普通态 / 选中态 / 拖拽态 配色、边框、高亮透明度
- 片段排布对齐规则

**验收标准**
全品类轨道（视频/字幕/文字/音频）视觉风格统一，无大小不一、样式杂乱问题。

---

## 二、P0 任务组：基础编辑能力闭环

### 任务4：实现片段基础操作 — 删除 / 分割 / 复制 / 粘贴

**需求描述**
1. 完成轨道片段四大基础操作：删除、分割、复制、粘贴
2. 每次操作完成后，轨道自动 relayout，保持片段磁吸紧贴、无多余空隙、无重叠
3. 删除已有 store 层实现，需要补齐 UI 调用路径（context menu 或按钮激活）
4. 分割需要新增 store 层实现：在 playhead 位置将片段一分为二

**验收标准**
- 四项操作功能可用、响应正常
- 操作后轨道自动规整布局，不出现空白间隙、不互相覆盖

### 任务5：时间轴基础滑动 + 缩放定型

**需求描述**
1. 时间轴横向滑动浏览流畅，基础惯性滚动可用
2. 时间轴放大/缩小缩放功能正常
3. 缩放比例均匀，锚点跟随手指合理

**验收标准**
- 滑动不卡顿、拖拽跟手
- 缩放无级调节正常，不跳帧、不错乱坐标

### 任务6：统一轨道选中态 / 高亮 / 间距布局

**需求描述**
1. 统一所有片段选中后的边框、高亮、遮罩样式
2. 规范轨道整体边距、片段左右留白
3. 选中、未选中、编辑状态视觉区分清晰

**验收标准**
选中反馈明确，UI 整齐规整，全轨道样式统一无违和。

## 三、本轮暂缓不做

1. 主轨道左右拉伸精细交互、自动滚动、松手磁吸细节打磨
2. 转场、滤镜、调色等 V2 进阶功能
3. 云端资产、个人素材二期功能
4. 各类边缘 Case、极致拖拽手感、动画过渡细节

---

## 四、开发执行顺序

1. 字幕轨道文字预览渲染
2. 音频波形可视化
3. 全轨道视觉样式统一
4. 片段：删除 → 分割 → 复制 → 粘贴
5. 时间轴滑动 + 缩放手感定型
6. 统一选中态 / 高亮 / 间距
